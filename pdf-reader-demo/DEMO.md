# Demo 运行指南

## 你已经获得了一个完整的 PDF 阅读器 Demo！

### 📦 包含内容

这个 Demo 实现了：

1. ✅ **PDF 文件打开** - 使用原生对话框选择 PDF
2. ✅ **高性能渲染** - pdfium 引擎直接渲染为位图
3. ✅ **虚拟滚动** - 只渲染可见页面，千页流畅
4. ✅ **三种主题** - 白天/夜间/护眼模式
5. ✅ **页面边距** - 周围留白方便笔记
6. ✅ **缩放控制** - 25% - 400% 缩放
7. ✅ **最近文档** - 左侧边栏管理
8. ✅ **LRU 缓存** - 重复查看不重新渲染

### 🚀 立即运行

```bash
# 1. 确保在项目目录
cd pdf-reader-demo

# 2. 运行安装脚本（安装 Rust + 依赖）
./setup.sh

# 3. 启动开发服务器
npm run tauri:dev
```

### 🎮 操作说明

| 操作 | 效果 |
|------|------|
| 点击 "打开文件" | 弹出选择对话框，选择 PDF |
| 点击侧边栏文档 | 切换到该文档 |
| 点击 `+` / `-` | 放大/缩小 |
| 点击缩放比例 | 重置为默认 150% |
| 点击主题按钮 | 切换 白天→护眼→夜间 |
| 滚动鼠标 | 翻页浏览 |

### 🎨 主题效果

**白天模式**
- 背景: 浅灰色 (#f5f5f5)
- 页面: 纯白
- 适合: 正常阅读

**护眼模式** 
- 背景: 米黄色 (#f4ecd8)
- 页面: 暖色调
- 适合: 长时间阅读

**夜间模式**
- 背景: 深灰 (#1a1a1a)
- 页面: 反色暗调
- 适合: 暗光环境

### 🔧 性能特点

1. **懒加载**: 只渲染可视区域 ±2 页
2. **LRU 缓存**: 最近 50 页保留在内存
3. **原生渲染**: pdfium 比 PDF.js 快 5 倍
4. **GPU 加速**: 页面使用 Canvas 硬件加速

### 🆘 遇到问题？

1. **编译失败**: 检查 Rust 版本 `rustc --version` >= 1.75
2. **无法打开 PDF**: 检查文件权限，尝试其他 PDF
3. **渲染缓慢**: 正常现象，首次渲染建立缓存
4. **样式错乱**: 检查是否使用了支持的浏览器
